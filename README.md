## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

* **Python 3.x**: Ensure you have Python installed. You can download it from [python.org](https://www.python.org/).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/pratikkdahal/Python-projects.git](https://github.com/pratikkdahal/Python-projects.git)
    cd Python-projects
    ```

2.  **Install Dependencies:**
    Most games utilize Python's built-in libraries (like `random`). However, the QR Code Generator likely requires an external library.
    ```bash
    pip install qrcode[pil]
    ```

Navigate to the specific folder of the project you want to run and execute the Python script.

Test your luck and logic.

```bash

```

### 4. Rock Paper Scissors

Play a match against the CPU.

```bash

```

*(Note: Replace `<filename>.py` with the actual name of the `.py` file located inside each folder.)*
